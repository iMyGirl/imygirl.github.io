## Git github
+ [安装与使用](https://github.com/iMyGirl/Git-Github)

+ [日常操作使用]
```
# 下载仓库
git clone <SSH链接>
# 在下载仓库中改动后上传
git add .
git commit -m "添加说明"
git push # 完成上传


# 其他命令
git status # 查看当前git状态

```
